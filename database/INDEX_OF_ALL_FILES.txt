========================================
HOW TO USE (Simple Steps):
========================================

STEP 1: INSTALLATION
--------------------
For phpMyAdmin users:
1. Open "00_READ_ME_FIRST_PHPMYADMIN.txt"
2. Follow the 3 steps inside
3. Run "01_INSTALL_FOR_PHPMYADMIN.sql"

OR use the visual launcher:
1. Open "CLICK_HERE_TO_START.html" in your browser
2. Click on installation files
3. Follow instructions


STEP 2: VERIFY
--------------
Run "VERIFY_INSTALLATION.sql" to check:
- Tables created?
- Views created?
- Procedures created?
- Sample data inserted?


STEP 3: USE REPORTS
-------------------
After successful installation:
1. Open "USER_ACTIVITY_REPORT_QUERIES.sql"
2. Copy any report query
3. Paste in phpMyAdmin
4. Run and see results!

========================================
RECOMMENDED WORKFLOW:
========================================

Day 1 - Installation:
1. Read 00_READ_ME_FIRST_PHPMYADMIN.txt
2. Run 01_INSTALL_FOR_PHPMYADMIN.sql in phpMyAdmin
3. Run VERIFY_INSTALLATION.sql to confirm
4. Test with: SELECT * FROM v_today_activity;

Day 2+ - Using Reports:
1. Open USER_ACTIVITY_REPORT_QUERIES.sql
2. Choose report you need
3. Copy query
4. Run in phpMyAdmin
5. Export results if needed

Monthly Maintenance:
1. Run: CALL sp_cleanup_old_activity_logs();
2. Removes logs older than 90 days
3. Keeps database fast


========================================
NEED HELP?
========================================

If you get errors:
1. Check INSTALLATION_GUIDE.md
2. Look at troubleshooting section
3. Verify you ran installation first
4. Make sure you're in 'sokapptest' database

Common Issues:
❌ "PROCEDURE does not exist" → Install first!
❌ "Table doesn't exist" → Run installation
❌ "PROCEDURE already exists" → Already installed, just use reports!
❌ No data showing → Normal if no activity today
